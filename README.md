# Browser ONNX OCR Lab

A fully in-browser OCR playground that compares preprocessing pipelines and model variants in real time.

## Why this is portfolio-worthy
- Directly shows ML engineering decisions and tradeoffs.
- No backend dependency: privacy-friendly edge inference.
- Great demo value for interviews.

## MVP scope
- Drag/drop image
- Run 3 preprocessing pipelines
- ONNX inference in browser
- Compare confidence and latency

## Tech stack
- Web: TypeScript + Vite
- Inference: ONNX Runtime Web
- Visualization: lightweight chart components
